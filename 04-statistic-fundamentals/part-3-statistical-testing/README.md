# Part 3 - Statistical Testing

Statistical testing helps us decide whether an observed difference is likely meaningful or just random chance.

## Main Questions

| Concept | Question Answered |
|---|---|
| Hypothesis Testing | Is there enough evidence that two groups are different? |
| P-value | How surprising is this result if there is actually no difference? |
| A/B Testing | Did version B perform differently from version A? |

## Why This Matters

Data Scientists use statistical testing before making decisions from experiments.

Examples:

- Old homepage vs new homepage
- Existing pricing vs new pricing
- Current teaching method vs new teaching method
- Version A email subject vs version B email subject

## Key Idea

A low p-value gives stronger evidence that the observed difference is not just random chance.

Statistical significance does not always mean business significance.
