# Part 2 - Distribution and Probability

Distribution and probability help us understand uncertainty.

## Main Questions

| Concept | Question Answered |
|---|---|
| Distribution | What shape does the data have? |
| Normal Distribution | Is the data shaped like a bell curve? |
| Probability | How likely is an event? |
| Conditional Probability | How likely is an event given another event? |
| Outlier | Is this value unusually far from the rest? |

## Why This Matters

Many machine learning concepts depend on probability and distribution.

Examples:

- Spam detection
- Fraud detection
- Recommendation systems
- Customer conversion prediction
- Outlier detection

## Key Idea

Machine learning does not usually give absolute answers.

It often gives probabilities.

Example:

```text
Spam probability = 0.95
```

This means the model thinks there is a 95% chance the email is spam.
